Bean Boutique Coffee Shop — Front-End Website

This project is a fully responsive front-end website for a coffee shop, developed as part of the L4DC Front-End Website Development module at Daffodil International Academy. The site is built using HTML, CSS, and JavaScript, focusing on clean structure, W3C standards, accessibility, responsive layouts, cross-browser support, and integration of third-party plugins. The design reflects a real-world e-commerce front-end with multiple pages, mobile navigation, and product showcase features.

The website includes the following pages: Home, Coffee Selection, Brewing Equipment, Events & Workshops, Special Offers, and Cart. Each page is connected with a consistent navigation bar and layout system to ensure user-friendly browsing. Responsive design is achieved through CSS Grid, Flexbox, and Media Queries, allowing the site to adapt to screens from 320px mobile devices to full desktop layouts. The hamburger menu enables mobile navigation, and product grids adjust to maintain spacing and readability.

W3C validation was performed for both HTML and CSS to ensure semantic markup and standard compliance. Accessibility tests were carried out using the WAVE tool, checking contrast ratios, semantic structure, form label assignment, and screen-reader compatibility. Improvements were made based on results, including adjusting contrast, adding missing labels, and slowing down slideshow transitions to assist users with disabilities. The website supports users with low-vision through text zoom, screen reader compatibility, and improved color contrast.

Cross-browser compatibility was tested in Google Chrome and Mozilla Firefox. Both browsers displayed the website consistently, though minor differences were noted in default font rendering and video player controls due to engine variations (Chrome uses Blink, Firefox uses Gecko). The website also underwent responsive testing for mobile views, with successful results across major breakpoints. However, minor layout issues may occur on extremely small devices under 300px width, and these are noted for future correction.

The project integrates plugins and external tools to expand functionality. Google Maps JavaScript API is used to show nearby locations of Bean Boutique, a JavaScript search filter plugin allows users to filter coffee beans by category, and a simple math-based captcha modal is added for basic spam protection in forms. While suitable for general use, the captcha is not recommended for secure financial transactions. Bootstrap was used selectively for UI components, contributing to faster development, consistent layout structure, and built-in responsiveness.

Version control was implemented using Git and GitHub, allowing iterative development and tracking of changes. GitHub provides commit history, backup, code safety, and collaboration potential for team-based work. Understanding Git branching and version control workflow is part of the learning outcome of this project.

From a technological perspective, CSS Grid and Flexbox were the most valuable tools in creating adaptive layouts across the website. Media queries were essential for responsive breakpoints, while semantic HTML strengthened accessibility. JavaScript enhanced interactivity with dynamic components such as menu toggles, slideshows, and search filters.

Future improvements include connecting the site to a backend with a database for real e-commerce functionality, adding real-time form validation, building a cart management system with JavaScript or API integration, compressing media for performance, and applying lazy loading for images. Payment system integration through gateways like Stripe or PayPal would be a next step for real-world deployment.

In conclusion, this project created a user-friendly, standards-compliant, and mobile-responsive website for Bean Boutique Coffee Shop using HTML, CSS, and JavaScript. Through W3C validation, accessibility testing, browser testing, and responsive design, the project demonstrates practical knowledge of front-end development. The outcome reflects professional front-end practices and prepares the website for future expansion into a full-stack e-commerce solution.
